Quotes API

API is running

Available Endpoints

GET /api/quotes
Get all quotes from the database
GET /api/quotes/random
Get a random quote from external API
GET /api/quotes/{id}
Get a specific quote by ID
POST /api/quotes
Save a new quote to the database
POST /api/quotes/random/save
Fetch a random quote from API and save it to database
DELETE /api/quotes/{id}
Delete a quote by ID

Quotes REST API | Built with Spring Boot
